## 🚀 Punti salienti del primo rilascio

### 🔐 Supporto per più protocolli VPN

Questa prima versione include il supporto per **3 tipi di protocollo VPN**:

- **OpenVPN**
- **VeRay**
- **WireGuard**

Questo consente agli utenti di scegliere il protocollo che meglio si adatta al loro profilo di connessione, alle esigenze di compatibilità e alle aspettative di prestazioni.

---

## 🌍 Server VPN gratuiti e premium

MyVPN offre una vasta gamma di server VPN, tra cui:

- **Server VPN gratuiti**
- **Server VPN premium** disponibili tramite **abbonamento**

Gli utenti possono iniziare con server gratuiti e passare a un’esperienza premium per accedere a più posizioni, migliori prestazioni e opzioni esclusive.

## 🚀 Destaques do lançamento inicial

### 🔐 Suporte para vários protocolos VPN

Esta primeira versão inclui suporte para **3 tipos de protocolo VPN**:

- **OpenVPN**
- **VeRay**
- **WireGuard**

Isso permite que os usuários escolham o protocolo que melhor se adapta ao seu perfil de conexão, às necessidades de compatibilidade e às expectativas de desempenho.

---

## 🌍 Servidores VPN gratuitos e premium

MyVPN oferece uma ampla gama de servidores VPN, incluindo:

- **Servidores VPN gratuitos**
- **Servidores VPN premium** disponíveis por **assinatura**

Os usuários podem começar com servidores gratuitos e fazer upgrade para uma experiência premium para acessar mais locais, melhor desempenho e opções exclusivas.

## 🚀 Initial Release Highlights

### 🔐 Multiple VPN Protocol Support
This first version includes support for **3 VPN protocol types**:

- **OpenVPN**
- **VeRay**
- **WireGuard**

This allows users to choose the protocol that best fits their connection profile, compatibility needs, and performance expectations.

---

## 🌍 Free and Premium VPN Servers
MyVPN offers a wide range of VPN servers, including:

- **Free VPN servers**
- **Premium VPN servers** available through **subscription**

Users can start with free servers and upgrade to a premium experience for access to additional locations, better performance, and more exclusive options.
